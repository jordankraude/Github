Overview

Purpose

My goal for my website is to be a conventional portfolio for myself. I don’t have the goal to attract a lot of user’s to the page for an artificial or business reason. The main reason is to showcase my talents, skills, and knowledge that I have gained throughout my college education. As such I have a resume with my past job experience, I have some pages showing some of my ability to code javascript. Lastly, I have links from the main web pages to the important sites that employers would need to go to in order to see all the rest of my personal projects in python, C, Clojure, etc. I have ways for those who see the website to get back in touch with me if they are interested and or send me files. Later, when I have a personal server, I will expand this webpage to have some tools that others can come and use such as a file uploader and sender (my own personal). To summarize, shorterm; I see this webpage being more of a resume where only a select few will visit (mainly employers who I have given the link to). Long Term, as I can expand and add onto the webpage, then I can start making it more known so others will want to visit.

Audience

Future employers.
